This folder contains a small **HTML/JS dashboard** used to preview the InvoWise API locally.

- KPI cards (YTD)

- Dark/light theme

Start FastAPI:



The frontend calls:

- `GET /health`
- `GET /invoices`
- `GET /analytics`
- `GET /reports/export`
